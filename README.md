# Application State (Demo Application)

A fairly simple application to illustrate different ways to store application state.
Uses minimum classes for fast build. It's a hybrid application with Hook examples included.

## Installation and run

```
npm install
npm start
```
## About
You can fork this project and utilize the fault free configuration.
```
React (16.8.0).
State Management: Redux (4.0.1), React Context.
Backend storage: None (Just local and application state).
CSS/SCSS: Utilizes grid.
Webpack.
Babel.
```
